# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-01-13

### Added
- Initial release of Airforce Quiz System
- Student dashboard with quiz overview
- Interactive quiz interface with timer
- 4 aviation safety quiz topics
- Score calculation and tracking
- Quiz review functionality
- Group assignment submission system
- Admin dashboard with analytics
- Question management interface
- Student results and grading pages
- Group grading functionality
- Authentication system with JWT
- Role-based access control (Student/Admin)
- React Query caching and optimization
- Supabase file storage integration
- Auto-group creation on submission
- Cache invalidation for real-time updates
- Admin statistics endpoint
- Professional documentation

### Features
- **Quiz System**
  - Timed quizzes (10 minutes)
  - Multiple choice questions
  - Server-side scoring
  - Answer review mode
  - Progress tracking

- **Group Assignments**
  - PDF file uploads
  - Group-based collaboration
  - Admin grading interface
  - Feedback system

- **Admin Tools**
  - Question CRUD operations
  - Student results export
  - Group management
  - Grading dashboard
  - System analytics

- **Security**
  - JWT authentication
  - Password hashing (bcryptjs)
  - Role-based routing
  - HTTPOnly cookies
  - Server-side validation

### Fixed
- TypeScript compilation errors
- React 19 hook imports
- Implicit any types
- Form submission handling
- Cache invalidation timing

### Technical Improvements
- Upgraded to React 19
- Next.js 16 with App Router
- Tailwind CSS 4
- TypeScript strict mode
- MongoDB connection pooling
